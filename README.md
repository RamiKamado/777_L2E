# 7x7 ラスト2エッジ 手順探索ツール

7x7ルービックキューブの「ラスト2エッジ」(UF/UBに残った崩れたエッジ)を揃える手順を、
指定した最大手数・持ち替え回数の範囲で双方向探索(meet-in-the-middle)するツールです。

## 使い方（GitHub Pagesで公開する場合）

1. このリポジトリの Settings → Pages を開く
2. Source を「Deploy from a branch」、Branch を「main」/「/(root)」に設定して Save
3. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` でアクセス可能になります

## ローカルで試す場合

`index.html` をブラウザで直接開くだけで動作します（サーバー不要の単一HTMLファイルです）。

## 使用手

`U2, R, Rw, 3Rw, 4Rw, Lw, 3Lw, x`（x2は不使用）

## 注意事項

- 最大手数を大きくすると探索に時間・メモリがかかります（14手を超えるとブラウザのメモリ上限に達し探索が完了しない場合があります）。
- UB面は「白・青」、UF面は「白・緑」のピースのみで構成されるのが物理的に正しい組み合わせです。
